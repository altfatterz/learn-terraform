### Packer
Use Hashicorp Packer to create a VM image which has Java pre-installed

Install Packer

```bash
$ brew tap hashicorp/tap
$ brew install hashicorp/tap/packer
```

Install the Azure Packer plugin

```bash
$ packer plugins install github.com/hashicorp/azure
```

Fill out the missing pieces in the ubuntu.json file

```bash
$ export SUBSCRIPTION_ID=$(az account show | jq -r .id)
$ az ad sp create-for-rbac --name="packer-create-vm" --role Contributor --scopes /subscriptions/$SUBSCRIPTION_ID --query "{ client_id: appId, client_secret: password, tenant_id: tenant }"
```

Build the image:

```bash
$ packer build ubuntu.json
```
